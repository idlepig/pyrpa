RPA Library from SeleniumLibrary
=====================================

src/rpa/keywords
-------------------

这个文件夹下多个功能模块 **crawler.py** 是爬虫相关的， **data.py** 是操作数据的模块， **file.py** 是文件相关的模块

src/rpa/utils 和 src/rpa/utils 和 src/rpa/base
---------------------------------------------------

这3个文件夹框架必要的一些设置

src/rpa/__init__.py
------------------------

keywords下多个功能的调度中心，以实现自动调用keywords的类和类里面的方法
